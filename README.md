# 住宅エネルギーシミュレーター

太陽光発電・蓄電池の経済効果を無料でシミュレーションできるWebアプリケーションです。

## アプリ概要

「エネがえる」に類似した日本の住宅向けエネルギーシミュレーションアプリです。
太陽光パネルや蓄電池を導入した場合の電気代削減効果・売電収入・長期経済効果を
地域別・屋根条件別に詳細計算します。

## 機能一覧

- **太陽光発電シミュレーション**
  - 地域別・方位別・傾斜角別の年間・月別発電量計算
  - 10地域（北海道〜沖縄）対応
  - 屋根方位8方向・傾斜角5種類対応

- **蓄電池効果計算**
  - 蓄電池容量別の自家消費率シミュレーション
  - 3〜20kWhの容量範囲に対応

- **経済効果計算**
  - 売電収入（FIT：固定価格買取制度）
  - 電気代削減額
  - 月別・年別の詳細内訳

- **長期経済効果予測（20年間）**
  - パネル経年劣化（0.5%/年）考慮
  - 電気代上昇（1.5%/年）考慮
  - 投資回収年数の算出
  - 累計経済効果グラフ

- **グラフ・チャート表示**
  - 月別発電量・消費電力量比較棒グラフ
  - 月別電気代比較棒グラフ
  - 20年間累計収支折れ線グラフ

## 計算ロジック説明

### 年間発電量
```
年間発電量(kWh) = パネル容量(kW) × 地域係数 × 方位係数 × 傾斜角係数
```

### 地域係数（南向き30°基準）
| 地域 | 係数 |
|------|------|
| 沖縄 | 1500 |
| 四国 | 1420 |
| 九州 | 1400 |
| 中部 | 1380 |
| 近畿 | 1350 |
| 関東 | 1320 |
| 中国 | 1300 |
| 東北 | 1150 |
| 北陸 | 1100 |
| 北海道 | 1050 |

### 自家消費率
| 条件 | 自家消費率 |
|------|-----------|
| 蓄電池なし | 30% |
| 蓄電池〜5kWh | 50% |
| 蓄電池5〜10kWh | 62% |
| 蓄電池10kWh以上 | 70% |

### 経済効果
- 売電収入 = 余剰電力量(kWh) × 16円/kWh（FIT2024年度）
- 電気代削減 = 自家消費量(kWh) × 30円/kWh

### 初期費用（未入力時）
- 太陽光パネル：20万円/kW
- 蓄電池：15万円/kWh

### 長期シミュレーション
- パネル経年劣化：0.5%/年
- 電気代上昇率：1.5%/年

## カスタマイズ方法

### 計算パラメーターの変更
`src/utils/calculations.ts` の定数を編集します：

```typescript
const FIT_PRICE = 16;                    // FIT売電価格（円/kWh）
const ELECTRICITY_PURCHASE_PRICE = 30;   // 電力購入単価（円/kWh）
const ELECTRICITY_ESCALATION = 0.015;    // 電気代上昇率（1.5%/年）
const PANEL_DEGRADATION = 0.005;         // パネル劣化率（0.5%/年）
```

### 地域係数の変更
`src/utils/calculations.ts` の `REGIONAL_COEFFICIENT` オブジェクトを編集します。

### UIカラーの変更
`tailwind.config.js` の `theme.extend.colors` を編集します。
